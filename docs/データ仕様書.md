## 1. 概要

### 1.1 目的
このドキュメントは、img-resizerプロジェクトで使用されるデータモデル、データ構造、データ型、制約、検証ルールを定義します。

### 1.2 対象データ

- DTO（Data Transfer Object）
- 設定データ
- ファイル形式

## 2. エンティティ定義

### 2.1 ImageFile エンティティ

画像ファイルを表すドメインエンティティです。

**名前空間**: `ImgResizer.Domain.Entities`

**定義**:
```csharp
public class ImageFile
{
    public string FilePath { get; set; } = string.Empty;
    public string FileName { get; set; } = string.Empty;
    public string Extension { get; set; } = string.Empty;
    public long FileSize { get; set; }
    public DateTime CreatedAt { get; set; }
}
```

**プロパティ詳細**:

| プロパティ名 | 型 | 必須 | 説明 | 制約 |
|------------|-----|------|------|------|
| FilePath | string | 必須 | ファイルの完全パス | 最大長: 260文字（Windows）、空文字列不可 |
| FileName | string | 必須 | ファイル名（拡張子含む） | 最大長: 255文字、空文字列不可 |
| Extension | string | 必須 | ファイル拡張子 | `.jpg`, `.jpeg`, `.png`, `.gif`, `.bmp` のいずれか |
| FileSize | long | 必須 | ファイルサイズ（バイト） | 0以上、52,428,800（50MB）以下 |
| CreatedAt | DateTime | 必須 | ファイル作成日時 | - |

**ビジネスルール**:
- `FilePath`は`FileName`と`Extension`から構成される
- `Extension`は大文字小文字を区別しない
- `FileSize`は0より大きい値でなければならない

## 3. DTO（Data Transfer Object）定義

### 3.1 ResizeImageRequest DTO

画像変換リクエストを表すDTOです。

**名前空間**: `ImgResizer.Application.DTOs`

**定義**:
```csharp
public class ResizeImageRequest
{
    public string FilePath { get; set; } = string.Empty;
    public string? ResizeMode { get; set; }
}
```

**プロパティ詳細**:

| プロパティ名 | 型 | 必須 | デフォルト値 | 説明 | 制約 |
|------------|-----|------|------------|------|------|
| FilePath | string | 必須 | - | 画像ファイルのパス | 空文字列不可、最大長: 260文字 |
| ResizeMode | string? | オプション | `"fit"` | 変換方式 | `"fit"` または `"crop"` のみ有効 |

**JSONスキーマ**:
```json
{
  "type": "object",
  "required": ["filePath"],
  "properties": {
    "filePath": {
      "type": "string",
      "description": "画像ファイルのパス",
      "maxLength": 260
    },
    "resizeMode": {
      "type": "string",
      "enum": ["fit", "crop"],
      "default": "fit",
      "description": "変換方式"
    }
  }
}
```

**バリデーションルール**:
1. `FilePath`は空文字列であってはならない
2. `FilePath`に `..` や `~` などの危険な文字列が含まれていないこと
3. `ResizeMode`は `"fit"` または `"crop"` のみ有効（大文字小文字を区別しない）
4. `ResizeMode`が省略された場合、デフォルトで `"fit"` が適用される

**使用例**:
```json
{
  "filePath": "C:\\images\\input\\photo.jpg",
  "resizeMode": "fit"
}
```

### 3.2 ResizeImageResponse DTO

画像変換レスポンスを表すDTOです。

**名前空間**: `ImgResizer.Application.DTOs`

**定義**:
```csharp
public class ResizeImageResponse
{
    public bool Success { get; set; }
    public string Message { get; set; } = string.Empty;
    public string? OutputPath { get; set; }
    public string? ResizeMode { get; set; }
    public string? ErrorCode { get; set; }
}
```

**プロパティ詳細**:

| プロパティ名 | 型 | 必須 | 説明 | 制約 |
|------------|-----|------|------|------|
| Success | bool | 必須 | 処理が成功したかどうか | - |
| Message | string | 必須 | 処理結果のメッセージ | 空文字列可 |
| OutputPath | string? | 条件付き | 変換後の画像ファイルパス | `Success`が`true`の場合に必須 |
| ResizeMode | string? | 条件付き | 使用された変換方式 | `Success`が`true`の場合に必須、`"fit"` または `"crop"` |
| ErrorCode | string? | 条件付き | エラーコード | `Success`が`false`の場合に必須 |

**JSONスキーマ（成功時）**:
```json
{
  "type": "object",
  "required": ["success", "message", "outputPath", "resizeMode"],
  "properties": {
    "success": {
      "type": "boolean",
      "example": true
    },
    "message": {
      "type": "string",
      "example": "画像を512×512に変換しました"
    },
    "outputPath": {
      "type": "string",
      "example": "C:\\images\\output\\photo_512x512.jpg"
    },
    "resizeMode": {
      "type": "string",
      "enum": ["fit", "crop"],
      "example": "fit"
    }
  }

```

**JSONスキーマ（エラー時）**:
```json
{
  "type": "object",
  "required": ["success", "message", "errorCode"],
  "properties": {
    "success": {
      "type": "boolean",
      "example": false
    },
    "message": {
      "type": "string",
      "example": "ファイルが見つかりません: C:\\images\\input\\photo.jpg"
    },
    "errorCode": {
      "type": "string",
      "example": "FILE_NOT_FOUND"
    }
  }
}
```

**使用例（成功時）**:
```json
{
  "success": true,
  "message": "画像を512×512に変換しました",
  "outputPath": "C:\\images\\output\\photo_512x512.jpg",
  "resizeMode": "fit"
}
```

**使用例（エラー時）**:
```json
{
  "success": false,
  "message": "ファイルが見つかりません: C:\\images\\input\\photo.jpg",
```

## 4. 設定データ定義

### 4.1 ImageResizeSettings クラス

アプリケーション設定を表すクラスです。

**名前空間**: `ImgResizer.Infrastructure.Configuration`

**定義**:
```csharp
public class ImageResizeSettings
{
    public string InputDirectory { get; set; } = string.Empty;
    public string OutputDirectory { get; set; } = string.Empty;
    public TargetSizeSettings TargetSize { get; set; } = new();
    public string[] AllowedExtensions { get; set; } = Array.Empty<string>();
    public long MaxFileSize { get; set; } = 52428800; // 50MB
    public PaddingColorSettings? PaddingColor { get; set; }
    public ImageQualitySettings? ImageQuality { get; set; }
}

public class TargetSizeSettings
{
    public int Width { get; set; } = 512;
    public int Height { get; set; } = 512;
}

public class PaddingColorSettings
{
    public int R { get; set; } = 0;
    public int G { get; set; } = 0;
    public int B { get; set; } = 0;
    public int A { get; set; } = 255;
}

public class ImageQualitySettings
{
    public int JpegQuality { get; set; } = 90;
    public int PngCompressionLevel { get; set; } = 6;
}
```

**プロパティ詳細**:

| プロパティ名 | 型 | 必須 | デフォルト値 | 説明 | 制約 |
|------------|-----|------|------------|------|------|
| InputDirectory | string | 必須 | - | 入力ディレクトリパス | 空文字列不可、有効なディレクトリパス |
| OutputDirectory | string | 必須 | - | 出力ディレクトリパス | 空文字列不可、有効なディレクトリパス |
| TargetSize | TargetSizeSettings | 必須 | Width: 512, Height: 512 | ターゲットサイズ | - |
| TargetSize.Width | int | 必須 | 512 | ターゲット幅 | 1以上、4096以下 |
| TargetSize.Height | int | 必須 | 512 | ターゲット高さ | 1以上、4096以下 |
| AllowedExtensions | string[] | 必須 | - | 許可された拡張子リスト | 空配列不可、最低1つの拡張子が必要 |
| MaxFileSize | long | 推奨 | 52428800 (50MB) | 最大ファイルサイズ（バイト） | 1048576 (1MB) 以上、104857600 (100MB) 以下 |
| PaddingColor | PaddingColorSettings? | オプション | R: 0, G: 0, B: 0, A: 255 | パディング色 | - |
| PaddingColor.R | int | オプション | 0 | パディング色（赤） | 0～255 |
| PaddingColor.G | int | オプション | 0 | パディング色（緑） | 0～255 |
| PaddingColor.B | int | オプション | 0 | パディング色（青） | 0～255 |
| PaddingColor.A | int | オプション | 255 | パディング色（透明度） | 0～255 |
| ImageQuality | ImageQualitySettings? | オプション | - | 画像品質設定 | - |
| ImageQuality.JpegQuality | int | オプション | 90 | JPEG品質 | 1～100 |
| ImageQuality.PngCompressionLevel | int | オプション | 6 | PNG圧縮レベル | 0～9 |

### 4.2 ImageResizeSettingsValidator クラス

設定値の検証を行うクラスです。アプリケーション起動時に設定値を検証し、無効な設定がある場合は起動を停止します。

**名前空間**: `ImgResizer.Infrastructure.Configuration`

**実装**: `IValidateOptions<ImageResizeSettings>`

**検証項目**:
- `OutputDirectory`: 必須チェック（空文字列不可）
- `TargetSize`: 必須チェック、Width/Heightが正の値であること
- `AllowedExtensions`: 必須チェック（1つ以上）、拡張子が`.`で始まること
- `MaxFileSize`: 正の値であること
- `PaddingColor`（オプション）: R/G/B/Aが0-255の範囲であること
- `ImageQuality`（オプション）: JpegQualityが1-100、PngCompressionLevelが0-9の範囲であること

**使用方法**:
```csharp
// Program.cs
builder.Services.AddOptions<ImageResizeSettings>()
    .Bind(builder.Configuration.GetSection("ImageResize"))
    .ValidateOnStart();

builder.Services.AddSingleton<IValidateOptions<ImageResizeSettings>, ImageResizeSettingsValidator>();
```

## 6. ファイル形式・構造

### 6.1 サポートされている画像形式

| 形式 | 拡張子 | MIMEタイプ | 説明 |
|------|--------|-----------|------|
| JPEG | `.jpg`, `.jpeg` | `image/jpeg` | 写真に適した形式、非可逆圧縮 |
| PNG | `.png` | `image/png` | 透明度をサポート、可逆圧縮 |
| GIF | `.gif` | `image/gif` | アニメーションGIFは最初のフレームのみ処理 |
| BMP | `.bmp` | `image/bmp` | Windows標準形式、非圧縮 |

### 6.2 ファイル名規則

#### 6.2.1 入力ファイル名
- 任意の有効なファイル名
- 拡張子は許可リストに含まれる必要がある
- 例: `photo.jpg`, `image_001.png`, `test-file.gif`

#### 6.2.2 出力ファイル名

**全体変換方式（fit）の場合**:
```
{元のファイル名}_512x512.{拡張子}
```

**例**:
- 入力: `photo.jpg` → 出力: `photo_512x512.jpg`
- 入力: `image.png` → 出力: `image_512x512.png`

**中央クロップ方式（crop）の場合**:
```
{元のファイル名}_512x512_crop.{拡張子}
```

**例**:
- 入力: `photo.jpg` → 出力: `photo_512x512_crop.jpg`
- 入力: `image.png` → 出力: `image_512x512_crop.png`

**規則**:
- 元のファイル名から拡張子を除いた部分を取得
- `_512x512`（または`_512x512_crop`）を付加
- 元の拡張子を維持

### 6.3 ファイルサイズ制限

| 項目 | 値 | 説明 |
|------|-----|------|
| 最小ファイルサイズ | 1バイト | 空ファイルは処理しない |
| 最大ファイルサイズ | 52,428,800バイト（50MB） | デフォルト値、設定で変更可能 |
| 推奨最大ファイルサイズ | 10,485,760バイト（10MB） | パフォーマンスを考慮 |

## 8. データ検証ルール

### 8.1 リクエストデータの検証

#### 8.1.1 FilePathの検証

| 検証項目 | ルール | エラーコード |
|---------|--------|------------|
| 必須チェック | 空文字列でないこと | `INVALID_REQUEST` |
| 形式チェック | 有効なファイルパス形式であること | `INVALID_REQUEST` |
| パストラバーサル | `..` や `~` などの危険な文字列が含まれていないこと | `INVALID_REQUEST` |
| 拡張子チェック | 許可リストに含まれる拡張子であること | `INVALID_FILE_FORMAT` |
| ファイル存在 | ファイルが存在すること | `FILE_NOT_FOUND` |
| ファイルサイズ | 最大ファイルサイズ以下であること | `FILE_TOO_LARGE` |

**検証コード例**:
```csharp
// 必須チェック
if (string.IsNullOrWhiteSpace(filePath))
{
    throw new ValidationException("ファイルパスが指定されていません");
}

// パストラバーサルチェック
if (filePath.Contains("..") || filePath.Contains("~"))
{
    throw new ValidationException("無効なファイルパスです");
}

// 拡張子チェック
var extension = Path.GetExtension(filePath).ToLower();
var allowedExtensions = new[] { ".jpg", ".jpeg", ".png", ".gif", ".bmp" };
if (!allowedExtensions.Contains(extension))
{
    throw new ValidationException($"サポートされていない画像形式です: {extension}");
}
```

#### 8.1.2 ResizeModeの検証

| 検証項目 | ルール | エラーコード |
|---------|--------|------------|
| 有効値チェック | `"fit"` または `"crop"` のみ有効 | `INVALID_REQUEST` |
| デフォルト値 | 省略時は `"fit"` を適用 | - |

**検証コード例**:
```csharp
var validModes = new[] { "fit", "crop" };
var resizeMode = request.ResizeMode ?? "fit";

if (!validModes.Contains(resizeMode.ToLower()))
{
    throw new ValidationException($"無効な変換方式です: {resizeMode}。有効な値は 'fit' または 'crop' です。");
}
```

### 8.2 設定データの検証

#### 8.2.1 必須項目の検証

| 項目 | 検証ルール | エラーコード |
|------|----------|------------|
| InputDirectory | 空文字列でないこと | `INVALID_CONFIGURATION` |
| OutputDirectory | 空文字列でないこと | `INVALID_CONFIGURATION` |
| TargetSize | nullでないこと | `INVALID_CONFIGURATION` |
| AllowedExtensions | 空配列でないこと、最低1つの拡張子が必要 | `INVALID_CONFIGURATION` |

#### 8.2.2 値の範囲検証

| 項目 | 範囲 | エラーコード |
|------|------|------------|
| TargetSize.Width | 1以上、4096以下 | `INVALID_CONFIGURATION` |
| TargetSize.Height | 1以上、4096以下 | `INVALID_CONFIGURATION` |
| MaxFileSize | 1MB以上、100MB以下 | `INVALID_CONFIGURATION` |
| PaddingColor.R/G/B/A | 0～255 | `INVALID_CONFIGURATION` |
| ImageQuality.JpegQuality | 0～100 | `INVALID_CONFIGURATION` |
| ImageQuality.PngCompressionLevel | 0～9 | `INVALID_CONFIGURATION` |

**検証コード例**:
```csharp
if (settings.TargetSize.Width < 1 || settings.TargetSize.Width > 4096)
{
    throw new ConfigurationException("TargetSize.Widthは1以上4096以下である必要があります");
}

if (settings.MaxFileSize < 1048576 || settings.MaxFileSize > 104857600)
{
    throw new ConfigurationException("MaxFileSizeは1MB以上100MB以下である必要があります");
}
```

## 10. データ変換規則

### 10.1 ファイルパスの正規化

**規則**:
- 相対パスは絶対パスに変換する
- パス区切り文字を統一する（Windows: `\`, Linux/macOS: `/`）
- 末尾の区切り文字を削除する

**例**:
```
入力: "images/input/photo.jpg"
出力: "C:\\images\\input\\photo.jpg" (Windows)
```

### 10.2 拡張子の正規化

**規則**:
- 小文字に変換する
- 先頭のドット（`.`）を含める

**例**:
```
入力: ".JPG", "JPG", ".jpg"
出力: ".jpg"
```

### 10.3 出力ファイル名の生成

**全体変換方式（fit）の場合**:
```csharp
var fileName = Path.GetFileNameWithoutExtension(inputPath);
var extension = Path.GetExtension(inputPath);
var outputFileName = $"{fileName}_512x512{extension}";
```

**中央クロップ方式（crop）の場合**:
```csharp
var fileName = Path.GetFileNameWithoutExtension(inputPath);
var extension = Path.GetExtension(inputPath);
var outputFileName = $"{fileName}_512x512_crop{extension}";
```

## 11. データ整合性

### 11.1 整合性ルール

1. **ファイルパスとファイル名の整合性**
   - `FilePath`の最後の部分は`FileName`と一致する必要がある

2. **拡張子の整合性**
   - `Extension`は`FileName`の拡張子と一致する必要がある
   - `Extension`は`AllowedExtensions`に含まれる必要がある

3. **ファイルサイズの整合性**
   - `FileSize`は実際のファイルサイズと一致する必要がある
   - `FileSize`は`MaxFileSize`以下である必要がある

4. **レスポンスデータの整合性**
   - `Success`が`true`の場合、`OutputPath`と`ResizeMode`は必須
   - `Success`が`false`の場合、`ErrorCode`は必須

### 11.2 整合性チェック

**コード例**:
```csharp
// ファイルパスとファイル名の整合性チェック
var fileNameFromPath = Path.GetFileName(filePath);
if (fileNameFromPath != imageFile.FileName)
{
    throw new DataIntegrityException("ファイルパスとファイル名が一致しません");
}

// 拡張子の整合性チェック
var extensionFromFileName = Path.GetExtension(imageFile.FileName).ToLower();
if (extensionFromFileName != imageFile.Extension.ToLower())
{
    throw new DataIntegrityException("ファイル名と拡張子が一致しません");
}
```
